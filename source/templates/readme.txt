
TEMPLATES
-----------------------------------------------------------------------

Visual templates for imagevue, the slideshow, and the admin module.

Currently only provided templates for the slideshow. We will be
providing additional templates as soon as we create them or get them.

The creation of a "template" is fairly basic - It generally consists
of a background image, a startimage and a configuration file where
appropriate colors have been set to the bgcol and textcol settings.

We appreciate any suggestions or constributions of templates.
