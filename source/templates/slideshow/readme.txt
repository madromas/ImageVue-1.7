
SLIDESHOW TEMPLATES
-----------------------------------------------------------------------

We have provided a few templates for the slideshow, which simply come
 as config files with background images. IMAGES and AUDIO are NOt 
provided, so you will have to edit the config file and enter your 
own folder/path, and set audio to false or add an audio file.


HOW TO USE:
------------
Simply copy the files in any template into your root 
slideshow/imagevue folder. Open the new configuration file, edit
anything you need, (you will need to set a valid PATH atleast), and you
are ready to go. The link to your creation will be:

www.yourpage.com/gallery/slideshownew.php?configfile=newconfigfile.ini

If you want a shorter link, you can do one of the following:

#1
----------
Create a copy of slideshownew.php, and give it any name you like.
Open the new file in notepad, or any text editing tool, and edit line 12:
$config='newconfigfile.ini';

Your link will then be:
www.yourpage.com/gallery/yourcopy.php


#2
----------
You may also use the same procedure as above, but make sure you name
your file "index.php". Only do this if you dont want to use the
main imagevue gallery which uses index.php.
