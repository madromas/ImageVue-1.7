
SOURCE FILES
-----------------------------------------------------------------------

Source files for adding custom content or for curious peeping.
Be careful what you do, for there is no support or guarantee for s
uccess on moderating the SOURCE FILES.

Source files require Macromedia Flash8 Professional!

Files:
-------------- 
imageVue.fla - The latest source for the main gallery imageVue.swf
slideshow.fla - The sourcefile for the iamgevue slideshow component
admin.fla - Source for the admin section for imagevue
sfxlib.swf - A library of sound FX which is loaded into imageVue.
preload.fla - Sourcefile for the SWF that is used to preload an open images in popup window
backexample.fla - Example of the creation of a large background gradient loaded into imageVue
layerexample.fla - Example of the creation of a SWF layer which can be loaded ontop of the startimage in imageVue
sfxlib.fla - Library of sound effects utilized by imagevue and the slideshow
show.fla - Basic slideshow as an option for startimage
globalproto.as - A group of functions shared across multiple FLA sourcefiles
globalstyles.as - A group of textstyles shared across multiple FLA sourcefiles
