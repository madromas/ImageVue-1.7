
Batch Image Processing
-----------------------------------------------------------------------

Allthough the main images have no restrictions, the optimal image
dimensions for use on the internetis 640x480 or 800x600 (or vice versa
depending on aspect). Often, your images might originally be at sizes
from 1600x1200 or larger, which is most often to huge for publishing
on the internet - Too large to fit screen, and takes too long to load.

There are plenty of batch image processing applications out there, 
but we have provided batch droplets for Adobe Imageready. If you are
the ownder of a digital camera, most likely you allready have software
that came with the camera that resizes your images to more
'web-friendly' sizes.

If you server does not support GD2 or higher, and can not process
thumbnails automatically, you need to read the documentation 
here on how to create them manually:
http://www.imagevuex.com/index.php?p=33


More Information on the imageready droplets here:
http://www.imagevuex.com/index.php?p=46
