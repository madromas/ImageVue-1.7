
JAVASCRIPT
-----------------------------------------------------------------------

This folder contains a couple of javascripts required for the imagevue
components.

#swfobject_source.js
This javascript is used to embed SWF files into browser working around
the new ActiveX restrictions in Internet Explorer 7.
Reference: http://blog.deconcept.com/swfobject/

#window.js
This javascript is used to resize- and reposition popup windows.
