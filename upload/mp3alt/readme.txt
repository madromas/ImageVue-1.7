
MP3ALT
-----------------------------------------------------------------------

This is the alternative mp3 folder for adding your mp3 tracks that are
to be played if you have set the audio to read audio from imagefolders
The audioplayer will search this folder if an imagefolder does not have
an MP3 file. Simply add your own mp3 tracks. You may 
also set an optional folder for your mp3alt in the configuration
file(config.ini).
