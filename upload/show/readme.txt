
SHOW
-----------------------------------------------------------------------

This is the default folder for adding images, if you are using the
startimage slideshow. You may also set an optional folder for the
startimage slideshow images through the configuration(config.ini).

Startimage slideshow images must be 640x360.

The startimage slideshow must not be confused with the imagevue
slideshow component. They are individual files, and the startimage
slideshow is very basic. It goes under the name "show.swf" in the
main imagevue folder.
