The following folders are contained within imagevue17.zip:

1. UPLOAD/
-----------------
This folder contains the files you need to upload to your website. 
All you need to  do, is upload the entire folder and rename
it to something more appropriate if you like(Ex. "gallery");

Next step is to manage the folders under upload/content/.
All you need to do is create/delete/rename groupfolders and 
imagefolders. Then you can place images inside imagefolders and
create descriptions with the admin.

Everything else from menu creation to preview image(thumbnail) 
creation is done automatically by the gallery.

After everything is working, you might want to look into
CONFIGURATION to alter the style of your gallery or slideshow:

Your URLs will be:
www.yoursite.com/upload/
www.yoursite.com/slideshow.php
www.yoursite.com/upload/admin/


2. SOURCE
-----------------
This folder includes source files and fonts for moderating the FLA.
Also included, are droplets which require imageready/photoshop 
for resizing images.
